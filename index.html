Sudoku!

Your task is to fill out the "solveboard" function in index.js and create an algorithm that solves the board. The function is suppose to return the solution.

Rules

  • Each row should have the numbers 1 - 9
  • Each column should have the numbers 1 - 9
  • Each 3x3 marked with white or dark background should have the numbers 1 - 9
  • Number cant be repeated or you'll end up breaking one of the 3 other rules

Good luck!
